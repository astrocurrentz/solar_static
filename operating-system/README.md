# Operating System

Internal studio operating materials live here.

- `playbooks/` - Repeatable workflows and procedures.
- `decisions/` - Decision records and rationale.
- `finance/` - Finance planning, reporting, and references.
- `crm/` - Relationship and sales pipeline materials.
- `templates/` - Reusable operating templates.
- `notion-sync/` - Notion import, export, and sync materials.
